# Hangman Game

## Description

This is a simple text-based Hangman Game developed in Python.

The program randomly selects a word from a predefined list and allows the player to guess one letter at a time.

The player has six incorrect attempts to guess the complete word.

## Features

- Random word selection
- Six wrong attempts
- Input validation
- Displays guessed letters
- Console-based game

## Technologies Used

- Python
- random module

## Concepts Used

- while loop
- if-else
- strings
- lists

## How to Run

python hangman.py
